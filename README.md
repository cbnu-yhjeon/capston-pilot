## 📖 강의 개요

딥러닝 비전 기술을 활용한 대표적인 프로젝트를 수행함으로써, 지능화 교과목을 통해 배운 인공지능 지식을 바탕으로 **실무 능력**과 **문제해결 능력**을 배양합니다.

### 학습 목표

- 인공지능 기술을 활용한 **문제해결 역량** 배양
- 인공지능 관련 프레임워크 활용 및 **프로그래밍 실무능력** 강화
- 논문 리뷰를 통한 **논문 작성법**, 접근방법 및 분석기법 학습

### 문제해결 방법

2개의 딥러닝 비전 관련 문제가 제시되며, 팀을 구성하여 아래 단계로 프로젝트를 수행합니다.

```
1단계  →  강의를 통해 적용 기술에 관한 개념 설명과 소개
2단계  →  매주 멘토링과 주제발표를 통해 세부 기술 습득
3단계  →  토의, 질의응답, 피드백을 통해 프로젝트 완성
```
